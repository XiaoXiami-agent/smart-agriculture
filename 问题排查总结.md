# 智慧农业 App 开发文档

## 项目概述

智慧农业 App 是一款帮助农民和园艺爱好者管理作物、记录农事活动的微信小程序。通过结合天气数据，提供智能化的农事建议，让种植更简单高效。

---

## 功能模块

### 1. 首页（天气 + 农事建议）
- 显示当前位置天气
- 智能农事建议（根据温度、湿度、降雨）
- 天气预警提醒
- 3日天气预报
- 今日待办任务快捷入口

### 2. 作物管理
- 添加作物（12种常见作物类型）
- 作物列表展示
- 作物详情查看
- 生长阶段记录
- 养护日志

### 3. 任务管理
- 今日任务清单
- 任务完成/删除
- 农事日历（按日期查看任务）
- 天气关联提醒

### 4. 灌溉管理
- 土壤湿度监测
- 各作物灌溉状态
- 智能灌溉建议

### 5. 设置
- 位置设置
- 天气预警开关
- 消息通知设置
- 清除缓存

## 数据模型

### 作物 (crops)
```javascript
{
  _id: String,
  name: String,           // 作物名称
  variety: String,        // 品种
  icon: String,           // 图标
  plantDate: String,      // 种植日期
  expectedHarvest: String,// 预期收获日期
  stage: String,          // 生长阶段
  status: String,         // 状态：normal/warning/danger
  statusText: String,     // 状态文本
  days: Number,           // 种植天数
  notes: String,          // 备注
  createTime: Number      // 创建时间戳
}
```

### 任务 (tasks)
```javascript
{
  _id: String,
  cropId: String,         // 关联作物ID
  type: String,           // 任务类型：water/fertilize/spray/harvest
  typeIcon: String,       // 任务图标
  title: String,          // 任务标题
  date: String,          // 任务日期
  status: String,        // 状态：pending/done/skip
  weatherRelated: Boolean,// 是否与天气相关
  remark: String         // 备注
}
```

### 设置 (settings)
```javascript
{
  location: String,       // 当前位置名称
  locationId: String,    // 位置ID
  alerts: {
    frost: Boolean,      // 霜冻预警
    highTemp: Boolean,   // 高温预警
    heavyRain: Boolean,  // 暴雨预警
    drought: Boolean     // 干旱预警
  },
  notification: Boolean  // 消息通知
}
```

## 天气 API 字段参考

### 实时天气 (now)
```javascript
{
  temp: "26",           // 温度
  feelsLike: "28",      // 体感温度
  humidity: "85",        // 湿度
  windDir: "东南风",     // 风向
  windScale: "3",        // 风力等级
  text: "多云"          // 天气描述
}
```

### 天气预报 (forecast.daily)
```javascript
{
  fxDate: "2026-04-28", // 日期
  textDay: "大雨",       // 白天天气

  tempMin: "22",         // 最低温度
  humidity: "87",         // 湿度
  precip: "3.5"          // 降水量
}
```

## 农事建议逻辑

### 温度判断
| 温度范围 | 建议 |
|---------|------|
| < 5°C | 低温预警！注意防霜冻，盖保温膜 |
| 5-15°C | 气温较低，适合浇水但避免喷药 |
| 15-30°C | 最佳农作温度，可进行浇水施肥喷药 |
| > 35°C | 高温预警！避免中午作业，注意灌溉 |

### 湿度判断
| 湿度范围 | 建议 |
|---------|------|
| > 85% | 湿度较高，注意通风防病 |
| 40-85% | 湿度适中 |
| < 40% | 空气干燥，加强灌溉 |
